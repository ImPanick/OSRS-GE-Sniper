## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Environment Variables

- `NEXT_PUBLIC_API_URL` - Backend API URL (default: http://localhost:5000)

## Pages

- `/dashboard` - Main dashboard with top flips, dumps, and spikes
- `/volume-tracker` - Advanced volume tracker with filtering
- `/admin` - Admin panel for server management and updates

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Axios for API calls
- Lucide React for icons
